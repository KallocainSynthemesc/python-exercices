Écrivez un programme Python qui itère les entiers de 1 à 100. Mais pour les multiples de trois, imprimez "Fizz" à la place du nombre et pour les multiples de cinq, imprimez "Buzz". Pour les nombres qui sont des multiples de trois et de cinq, imprimez "FizzBuzz".

Loop exercise - FizzBuzz extended

J'ai étendu notre programme FizzBuzz pour imprimer "Fuzz" pour les multiples de sept et "Bizz" pour les multiples de onze. Nous répétons sans cesse des instructions très similaires. Ce n'est pas un bon style de programmation et il faut absolument le remanier. Pouvez-vous extraire le contrôle if (par exemple : if i % 3==0:) dans une fonction et l'appeler simplement dans notre boucle ?

                      def fizzBuzz():
                        i = 1
                        while i<=100:
                          output = '';
                          if i % 3 ==0:
                            output += 'Fizz'
                          if i % 5 == 0:
                            output += 'Buzz'
                          if i % 7 == 0:
                            output += 'Fuzz'
                          if i % 11 == 0:
                            output += 'Bizz'
                            
                          print(output if output != '' else str(i))
                          i += 1
  
                      fizzBuzz()
